# 项目描述
荔园学堂是一个mini版在线学习平台，主要分为两个端：
+ 用户端
+ 后台管理端

# 核心功能

+ 用户端：其核心业务主体就是学生，所有业务围绕着学生的展开
+ 管理端：其核心业务主体包括老师、管理员、其他员工，核心业务围绕着老师展开
